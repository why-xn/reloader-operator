ðŸš€ GETTING STARTED:

1. Verify the operator is running:

   kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "reloader-operator.name" . }}

2. Check the operator logs:

   kubectl logs -n {{ .Release.Namespace }} deployment/{{ include "reloader-operator.fullname" . }}-controller-manager -f

3. Create your first ReloaderConfig:

   kubectl apply -f - <<EOF
   apiVersion: reloader.stakater.com/v1alpha1
   kind: ReloaderConfig
   metadata:
     name: example-reloader
   spec:
     watchedResources:
       secrets:
         - my-secret
     targets:
       - kind: Deployment
         name: my-app
   EOF

4. Or use annotations on your existing workloads:

   kubectl annotate deployment my-app secret.reloader.stakater.com/reload=my-secret
